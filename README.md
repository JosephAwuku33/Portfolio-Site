## Tech Stack

- React
- Tailwind CSS
- Framer Motion
- React Router DOM


## Run Locally

Clone the project

```bash
  git clone https://github.com/JosephAwuku33/Portfolio-Site
```

Go to the project directory

```bash
  cd Portfolio-Site
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
